# Legal Term Bank (Preferred)

Use these terms consistently when drafting or revising clauses. Select jurisdiction-specific terms only when the contract or playbook requires them.

| Preferred term | Notes |
| --- | --- |
| Disclosing Party | NDA party providing Confidential Information. |
| Receiving Party | NDA party receiving Confidential Information. |
| Confidential Information | Use as a defined term; keep capitalization consistent. |
| Permitted Disclosure | Use for allowed disclosure exceptions. |
| Injunctive relief | Standard remedy language for confidentiality breaches. |
| Data Controller | GDPR role; pair with Data Processor. |
| Data Processor | GDPR role; pair with Data Controller. |
| Personal Data | GDPR term; use Personal Information where CCPA applies. |
| Data Subject | GDPR term for the individual. |
| Sub-processor | Hyphenate consistently. |
| Technical and organizational measures | Spell out once; introduce acronym only if needed. |
| Security Incident | Use for defined breach or incident events. |
| Breach notification | Use for incident notice obligations. |
| Audit rights | Use for inspection or audit provisions. |
| Data subject request | Use for access, deletion, or portability requests. |
| International transfer | Use for cross-border data transfers. |
| Standard Contractual Clauses (SCCs) | Use only where EU transfer mechanisms apply. |
| Binding Corporate Rules (BCRs) | Use only where EU transfer mechanisms apply. |
| Effective Date | Define the start date of the agreement. |
| Term | Define the duration of the agreement. |
| Termination | Use for ending the agreement; align with notice requirements. |
| Survival | Identify provisions that survive termination. |
| Assignment | Use for restrictions on transfer of rights/obligations. |
| Entire Agreement | Use for integration clause. |
| Severability | Use for invalid provision handling. |
| Notices | Use for formal notice requirements. |
| Dispute Resolution | Use for venue or arbitration procedure. |
| Arbitration | Use when arbitration is the chosen mechanism. |
| Indemnification | Use for indemnity obligations and scope. |
| Liability cap | Use for limitation of liability structure. |
| Governing law | Standard clause label. |
| Limitation of liability | Standard clause label. |
| Representations and warranties | Standard clause label. |
| Material breach | Use for termination triggers. |
| Force majeure | Standard clause label. |
